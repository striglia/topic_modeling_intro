The hints this time are pretty minimal (see section 2's hints.txt for some of these topics).

For more information on probabilistic topic modeling, I really like [this overview](http://www.scottbot.net/HIAL/?p=19113). It talks about a lot of different academic papers, links a few really cool applications, and overall will give you a great idea of what is possible with these approaches.

Having trouble installing gensim? Try downloading conda and (after it is set up) running `conda create -n gt_bigdata gensim scipy numpy; source activate gt_bigdata`. This should put you in a virtual environment with the packages installed.
